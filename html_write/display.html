프로퍼티값 키워드 설명
blcok block특성을 가지는 요소

none 해당 요소를 화면에 표시하지 않는다(공간마저사라진다)

block레벨 요소

block특성을 가지는 요소(block 레벨 요소 또는 block요소)

  • div
  • h1 ~ h6
  • p
  • ol
  • ul
  • li
  • hr
  • table
  • form

inline레벨 요소

inline특성을 가지는 요소
width,height,margin-top,margin-bottom property를 지정할 수 없다.

  1. span
  2. a
  3. strong
  4. img
  5. br
  6. input
  7. select
  8. textarea
  9. button

inline-block요소

block과 inline레벨 요소의 특징을 모두 갖는다.
inline레벨 요소와 같이 한 줄에 표현되면서 width,height,margin property를 모두 지정할 수 있다.
⋗inline-block 레벨 요소 뒤에 공백(엔터, 스페이스 등)이 있는 경우, 정의하지 않은 space(4px)가 자동 지정된다.

visibility

요소를 보이게 할 것인지 안 보이게 할 것인지를 정의한다. 즉, 요소의 렌더링 요보를 결정한다.

display 프로퍼티에 관한 표
프로퍼티값 키워드 설명
visible 해당 요소를 보이게 한다 (기본값)
hidden 해당 요소를 보이지 않게 한다. display: none;은 해당 요소의 공간까지 사라지게 하지만 visibility: hidden;은 해당 요소의 공간은 사라지지 않고 남아있게 된다.
collapse table 요소에 사용하며 행이나 열을 보이지 않게 한다.
none table 요소의 row나 column을 보이지 않게 한다. IE, 파이어폭스에서만 동작하며 크롬에서는 hidden과 동일하게 동작한다.
